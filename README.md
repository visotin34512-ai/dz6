 На вход программе поступают целые положительные числа. C помощью цикла реализовано их считывание, пока не встретится число 0. В теле цикла для каждого прочитанного числа вычисляется квадратный корень (с точностью до сотых) и значение выводится на экран (в столбик). С помощью словаря выполнено кэширование данных так, что при повторном вводе того же самого числа результат не вычислялся, а бралось ранее вычисленное значение (из словаря).
